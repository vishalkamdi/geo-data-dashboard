# Geo Data Dashboard

This is a React based Geo Data Dashboard project.
It shows project data in table and on map.

---

## Features

- Project list with pagination
- Handles more than 5000 records
- Search, filter and sort options
- Map view using Leaflet
- Table and map are connected

---

## Performance

Only 200 records are shown per page.
This helps the app to run smoothly
even with large data.

---

## Tech Used

- React (Vite)
- JavaScript
- Material UI
- Leaflet
